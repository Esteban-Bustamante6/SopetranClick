# SopetranClick
Sopetrán Click es una plataforma centralizada en Java Spring Boot que digitaliza el comercio turismo y gestión pública de Sopetrán, Antioquia. Con un diseño moderno en Dark Mode arquitectura limpia y base de datos relacional normalizada optimiza la movilidad y conecta a la comunidad con servicios locales mediante interfaces adaptativas y eficientes

🌲 Sopetrán Click"El corazón verde de Antioquia"Bienvenido al repositorio oficial de Sopetrán Click, una plataforma digital centralizada diseñada para digitalizar la economía local, potenciar el turismo y optimizar la movilidad y gestión pública del municipio de Sopetrán, Antioquia. 

Visión del ProyectoSopetrán Click nace como una herramienta integral desarrollada bajo un enfoque polymath, que combina arquitectura de software de alta calidad con una experiencia de usuario (UX) centrada en la identidad cultural de la región. Nuestro objetivo es cerrar la brecha digital en el municipio, centralizando en una sola plataforma: comercios, servicios, turismo y eventos públicos.

Arquitectura TécnicaEl proyecto sigue un enfoque monolítico modular utilizando el ecosistema de Java Spring Boot, garantizando una gestión robusta de los datos y un mantenimiento sencillo.Estructura en Capas (Separación de Responsabilidades)domain / model: Entidades JPA que definen el núcleo de los datos.repository: Abstracción de datos mediante interfaces JpaRepository.service: Lógica de negocio, validaciones y reglas operativas.controller: Gestión de rutas, peticiones HTTP y renderizado de vistas.

Frontend y UX/UIPriorizamos una interfaz moderna, elegante y de alto contraste mediante:Identidad: Tema oscuro (Dark Mode) predominante que resalta la fotografía de los paisajes y comercios locales.Experiencia: Animación de bienvenida inmersiva centrada en la iglesia municipal.Adaptabilidad: Diseño de interfaces específicas según el sector comercial:Restaurantes: Enfoque visual e ingredientes.Ferreterías: Buscadores avanzados con fichas técnicas y cotización rápida.Servicios (Barberías): Catálogo de estilos con calendario integrado para agendamiento.

Base de Datos RelacionalDiseñada bajo un modelo normalizado que soporta 5 ejes estratégicos, optimizada para minimizar valores NULL mediante el uso de tablas genéricas de servicios:Usuarios: Roles dinámicos (Turista, Comerciante, Administrador).Ejes Maestros: Alojamientos, Ecoturismo (veredas y rutas), Comercio Local, Entidades Públicas y Transporte.Gestión: Reservas, contactos directos y sistema de valoración (comentarios).

Tecnologías UtilizadasBackend: Java, Spring Boot.Frontend: HTML5, CSS3, JavaScript, Thymeleaf.Gestión de Datos: JPA / Hibernate.Proyecto desarrollado con pasión por la tecnología y amor por el corazón verde de Antioquia. 🌿
